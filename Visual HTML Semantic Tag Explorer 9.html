When to Use This Tag

Use <section> to group related content that has a thematic connection. Each section should ideally have a heading that describes its content.

‚úÖ Best Practices
  • Always include a heading (h1-h6) in each section
  • Use sections for thematically related content
  • Don't use section just for styling purposes
  • Consider using article for standalone content

Common Mistakes to Avoid

<!-- ‚ùå Don't use section without a heading --> <section> <p>Some content...</p> </section> <!-- ‚úÖ Do include a heading --> <section> <h2>About Us</h2> <p>Our company story...</p> </section>
‚ôø Accessibility Features

The <section> element provides important structural information to assistive technologies:

  • Creates landmark regions for screen readers
  • Enables skip navigation functionality
  • Improves content organization for users with cognitive disabilities
  • Supports ARIA roles and properties

ARIA Integration

<section aria-labelledby="features-heading"> <h2 id="features-heading">Key Features</h2> <p>Content about features...</p> </section>
